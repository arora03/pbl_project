SwinStego

Mitigating Texture Drift in Generative Steganography

"Hypothesis: Replacing Global Attention (ViT) with Hierarchical Shifted Window Transformers (Swin) reduces computational complexity from Quadratic \(O(N^2)\) to Linear \(O(N)\) while minimizing texture distortion and perceptual artifacts in stego images."

System Methodology

1. Input Processing

Cover image and secret message are combined into a single tensor and passed into the Swin Transformer backbone.

2. Shifted Windows

Attention is computed locally within small windows to preserve fine textures, and windows shift across layers to gradually build global context efficiently.

3. Spatial Transformer (STN)

Learns to correct geometric distortions such as rotation and scaling before message extraction, improving robustness against spatial attacks.

Results

PSNR 40.2 dB
SSIM 0.9732
Accuracy 99.39%
