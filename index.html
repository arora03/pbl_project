SwinStego

Mitigating Texture Drift in Generative Steganography

"Hypothesis: Replacing Global Attention (ViT) with Hierarchical Shifted Window Transformers (Swin) restores local texture fidelity, eliminates ghosting artifacts, and reduces computational complexity from Quadratic \(O(N^2)\) to Linear \(O(N)\)."

The Problem: Texture Drift

1. Visual Artifacts: Standard Vision Transformers (ViT) use global attention that "washes out" fine details. This results in "Texture Drift"—faint, blocky, grid-like ghosting artifacts in the stego-image.

2. Computational Cost: ViT has a complexity of \(O(N^2)\). This makes it impractical for high-resolution images (e.g., 1024x1024), limiting real-world application.

[Texture Drift / Checkerboard Artifacts]

Gap Analysis

"TransStego" (Xiao et al., 2025) solved CNN locality issues but introduced global blurring. We need a hybrid approach: Local Attention + Global Context.

System Methodology

1. Hierarchical Input

Cover Image + Secret Message concatenated into a tensor. Processed via Swin Blocks using W-MSA and SW-MSA.

2. Shifted Windows

Attention is computed locally within windows (preserving texture), then windows shift to share global context.

3. WGAN Critic

Uses Earth Mover's Distance instead of standard GAN loss to prevent mode collapse and ensure statistical indistinguishability.

Loss Function Strategy

$$L_{total} = \lambda_1 L_{pixel} + \lambda_2 L_{perceptual} + \lambda_3 L_{message} + \lambda_4 L_{adv}$$

Optimizing for Pixel Fidelity (L1), Human Perception (LPIPS), Message Recovery (BCE), and Adversarial Realism (WGAN).

Architectural Evolution

Feature Vision Transformer (ViT) Swin Transformer (Ours)
Attention Mechanism Global (All patches at once) Local (Shifted Windows)
Complexity Quadratic \(O(N^2)\) Linear \(O(N)\)
Visual Output Blocky / Grid Artifacts Content-Adaptive Texture
Scalability Low (Impractical for HD) High (Scalable to 1024px+)
